STUDENT GRADE GENERATOR

The project generates grades as follows once a mark has been input.
A > 79,
 B > 60 to 79,
  C > 59 to 49,
   D > 40 to 49,
    E > less 40. 
    Invalid = Grades below 0 and grades above 100.

The project has been build up using:
      -HTML
      -CSS
      -JS
